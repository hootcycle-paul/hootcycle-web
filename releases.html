Release Notes

Stay updated with the latest improvements to HootCycle

Version 2.1.4

May 2, 2025
  • Fixes and improvements:
    • 2.1.4 adds better background location and altitude tracking on Android, using a native Java plugin.
    • This also fixes an issue where the app could crash while approaching the beginning of a loaded route.

Version 2.0.2

April 21, 2024
  • Google Play Store release of HootCycle for Android
    • Android joins iOS as a supported platform with the same features.

Version 1.1.6

January 7, 2025
  • Apple App Store release of HootCycle for iOS
    • After a long phase of internal development and testing, HootCycle is now available on the App Store!
    • •     .fit file route support
    • •     Battery efficient ride tracking
    • •     Core ride metrics
    • •     Strava integration for activity uploads
